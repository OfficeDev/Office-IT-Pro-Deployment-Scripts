Visio for Office 365

Microsoft Visio Pro for Office 365, available as a subscription through Microsoft Office 365, includes the same capabilities as Visio Professional 2016. In addition, it allows each user to install Visio on up to five PCs running Windows 10, Windows 8, or Windows 7. It automatically installs the latest updates (both feature and security) for the duration of the subscription.


For more information see Visio for Office 365

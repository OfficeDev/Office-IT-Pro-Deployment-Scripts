Remote Client Configuration

Configure a remote client in order to be managed via WMI/Powershell

Powershell Configuration


Local Machine Configuration

  • 1. Open an instance of Powershell as an administrator
  • 2. Run the command "Set-ExecutionPolicy Bypass"

Remote Machine Configuration

  • 1. Open an instance of Powershell as an administrator
  • 2. Run the command "Set-ExecutionPolicy Bypass"
  • 3. Next, run the command "Enable-PSRemoting"

WMI Configuration

  • 1. Log into the remote computer and open command the prompt
  • 2. Run the command "dcomcnfg"
  • 3. In dcomconfig navigate to "Console Root > Computers >" right click "My Computer" then click "properties"
  • 4. Click the "COM Security" tab
  • 5. Under Access Permissions click "Edit Limits"
  • 6. Add the username that will used to access the computer remotely
  • 7. In the Permssions area click for the user you added make sure that the Remote Access permission is Allowed, click OK
  • 8. Repeat steps 5 and 6 for "Launch and Activation Permissions"
  • 9. In the Permissions area for the user you added make sure all permssions are allowed (Local Launch, Remote Launch, Local Activation, and Remote Activation), click OK
  • 10. In the "My Computer Properties" window click Apply then OK
  • 11. In the dcomcnfg window navigate to "Console Root > Computers > My Computer > DCOM Config" and right click on "Windows Management and Instrumentation" then click properties
  • 12. Click the "Security" tab
  • 13. Under "Launch and Activation Permissions" click edit
  • 14. Add the username that will used to access the computer remotely
  • 15. In the Permissions area for the user you added make sure all permssions are allowed (Local Launch, Remote Launch, Local Activation, and Remote Activation), click OK
  • 16. Under "Access Permissions" click the "Customize" radial button and then "Edit"
  • 17. Add the username that will used to access the computer remotely
  • 18. In the Permissions area for the user you added make sure that all permissions are allowed (Local Access, and Remote Access), click OK
  • 19. Under "Configuration Permissions" click edit
  • 20. Add the username that will used to access the computer remotely, click OK
  • 21. In the Windows Management and Instrumentation window click Apply then OK
  • 22. Close the dcomcnfg window
  • 23. Start Computer Management
  • 24. In Computer Management navigate to "Services and Applications" and right click on "WMI Control" then click properties
  • 25. In the WMI Control Properties window click the "Security" tab
  • 26. Click the "Root" folder then the Security button
  • 27. Add the username that will used to access the computer remotely
  • 28. In the Permissions area for the user you added make sure that all permissions are allowed (Execute Methods, Full Write, Partial Write, Provider Write, Enable Account, Remote Enable, Read Security, and Edit Security), click Apply then OK
  • 29. Expand the "Root" folder, then click the "CIMV2" folder and then the Security button.
  • 30. Repeat steps 27 and 28
  • 31. In the WMI Control Properties click OK
  • 32. Close the "Computer Managment" window
  • 33. Open the "Control Panel" and open "Windows Firewall"
  • 34. Click "Allow an app or feature through Windows Firewall"
  • 35. Click the checkbox next to "Windows Management Instrumentation (WMI)", click OK and close the "Windows Firewall" window
  • 36. Open regedit
  • 37. Navigate to either "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\ClickToRun\Configuration" or "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\16.0\Configuration", depending on which key exists
  • 38. Right click on the key then click permissions
  • 39. Add the username that will used to access the computer remotely
  • 40. In the Permissions area for the user you added make sure that all permissions are allowed (Full Control and Read), click Apply then OK
  • 41. Navigate to "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies" and repeat steps 38-40
  • 42. Close the "Registry Editor" window
  • 43. Restart remote computer

Silent Install

Determines whether or not the install will run silently.

Optional.

If Silent Install is set to Yes, the user sees no UI. No progress UI, completion screen, error dialog boxes, or first run automatic start UI are displayed. Additionally, the Display Level on the Options tab will be set to None and the Accept EULA option will be set to True regardless of their previous settings. If Silent Install is set to No, the user will see the UI. The progress UI, completion screen, error dialog boxes, and first run automatic start UI are displayed if applicable. Additionally, the Display Level on the Options tab will be set to Full and the Accept EULA option will be set to False regardless of their previous settings.

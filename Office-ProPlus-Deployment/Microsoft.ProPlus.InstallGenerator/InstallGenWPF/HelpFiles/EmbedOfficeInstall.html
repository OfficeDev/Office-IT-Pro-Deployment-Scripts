Embed Office installation files

This option will embed the Office installation files with the MSI or Executable.

The installation files are the files that are downloaded using the Office Deployment Tool (ODT). Using the local files allows you to install Office 365 ProPlus without having to download the files from the Internet.


Embedding the installation files enables you to distribute Office 365 ProPlus with a single file without the need for Internet Access. The single file will be large and will be over 1 GB in size.

Target Version

The Click-to-Run product version to install.

Optional.

If TargetVersion is not set, Click-to-Run updates to the most recent version from the Microsoft Click-to-Run source. If TargetVersion is set to empty (""), Click-to-Run updates to the latest version from the Microsoft Click-to-Run source.


TargetVersion can be set to an Office build number, for example, 15.1.2.3. When the version is set, Office attempts to transition to the specified version in the next update cycle. If you use Group Policy and the Office 2013 Administrative Template files (ADMX/ADML), you can set TargetVersion by using the Target Version policy setting. You can find this policy setting under Computer Configuration\Administrative Templates\Microsoft Office 2013 (Machine)\Updates.


For more information on the specific versions of Office Click-To-Run see Update Office 2013 or Office 365
